Hi! This is A Django boilerplate by Federico Varela You should see the toolbar here -->

Features:

Django Debug Toolbar

Django Debug Toolbar allows the developer to see the data associated with the request, such as SQL queries executed, request headers, cache, static files and more

Jupyter Notebooks

Jupyter Notebooks for debugging and tinkering with the ORM, just as the regular shell, but more comfortable and saves checkpoints of your document

Batch Files

The regular Django commands are abstracted behind batch files, allowing you to run them with a single word
You can also add call path/to/env after @ECHO OFF to activate your environment before running the command

Requirements

Project's requirements.txt file is included with all the dependencies ready

Migrations already ran

The SQLite database has the initial migrations already set up

Have I missed some functionality? Would you like to have something else included? Send me a pull request and I'll check it out
